# KeyRadar 무료 체험 정책

**버전 0.3** · 2026-05-30 (사업자 정보 확정에 따른 동기화)

본 문서는 [이용약관](./TERMS_ko.md) §3의 보충 설명입니다. 충돌 시 약관이 우선합니다.

## 한눈에 요약

| 항목 | 내용 |
|---|---|
| 기본 체험 기간 | **3일** (설치 직후부터) |
| 이전 설치 흔적 감지 시 | **1일**로 단축 |
| 체험 기록 저장 위치 | 사용자 기기 로컬에만 저장 |
| 운영자 서버 전송 | **없음** |
| 체험 종료 후 | 일부 기능 제한, 라이선스 구매로 해제 |

## 체험의 작동 방식
- 설치 직후 첫 실행 시점부터 3일간 모든 기능을 사용할 수 있습니다.
- 체험 시작일과 동일 기기의 이전 설치 표식은 KeyRadar 앱 로컬 영역에만 저장됩니다.
- 운영자(HAIB) 자체 서버는 체험 흐름에 관여하지 않습니다.

## 1일로 단축되는 조건
- 동일 기기에서 이전 KeyRadar 설치 흔적(로컬 표식)이 감지되면 체험은 1일로 단축됩니다.
- 이는 반복 체험 남용을 부분적으로 방지하기 위함입니다.

## 한계 (정직한 고지)
본 체험 제한 메커니즘은 로컬 표식에 기반하므로 **완벽하지 않습니다.** 다음 경우 우회될 수 있습니다.
- OS 재설치
- 기기 변경
- 사용자 계정 변경
- 로컬 표식 수동 삭제

이용약관 §7.5에 따라 사용자는 의도적 우회를 시도하지 않을 것에 동의합니다.

## 체험 종료 후
- 체험이 종료되면 KeyRadar의 일부 기능이 제한됩니다.
- 라이선스 구매([라이선스 정책](./LICENSE_POLICY_ko.md))로 모든 기능을 다시 사용할 수 있습니다.
- 체험 기간 중 수집·저장된 사용자 데이터(vault, refs, 로그 등)는 체험 종료 후에도 그대로 유지됩니다 — 라이선스 구매 후 즉시 이어서 사용 가능합니다.

## 향후 변경
- 현재 체험은 앱 내부 로컬 로직으로 처리됩니다. 향후 Polar의 체험 기능이나 운영자 자체 검증 서버를 도입할 경우, 본 정책을 사전 갱신·고지합니다.

## 변경 이력
- v0.3 (2026-05-30): 정책 패키지 전체 v0.3 동기화. 본 문서 내용 변경 없음.
- v0.2 (2026-05-30): 체험을 앱 로컬 로직으로 명시, 운영자 서버 비관여 명시, 우회 가능성 정직 고지.
