## 한 줄 요약
AI 코딩 도구가 흩뿌리는 API 키를 한 곳에 모으고, 쓸 때 감시하고 기록합니다. 막지 않습니다.

## 왜 만들었나
AI 코딩 에이전트(Cursor, Claude Code, Copilot 등)는 프로젝트마다 `.env` 파일을 만들거나 키를 묻거나 생성합니다. 결과적으로 사용자 PC 곳곳에 API 키가 흩어집니다. 어떤 키가 어디 있는지, 어느 키가 살아 있는지, 어느 키가 지난주에 갑자기 비용을 태우고 있는지 알기 어렵습니다.

KeyRadar는 그 흩어진 키들을 **로컬에 모으고**, **상태와 사용량을 한눈에 보여주고**, **이상 징후를 알려주는** 데스크톱 도구입니다.

## 무엇을 하는가
- 사용자가 선택한 폴더에서 `.env` 형태 파일을 찾아 후보로 보여줍니다.
- 사용자가 승인한 키만 로컬 암호화 vault로 옮깁니다.
- 원본은 즉시 삭제하지 않고 안전 보관(Safe Archive)합니다.
- 옮긴 자리에는 비밀 정보 없이 좌표만 적힌 `.keyradar.md` 파일을 남깁니다.
- 등록된 키의 상태(살아있음/죽음/권한 필요)와 사용량 트렌드를 보여줍니다.
- 비정상 급증(rapid burn) 등 이상 신호를 감지해 알립니다.

## 무엇을 하지 않는가 (정직한 한계)
- **클라우드 동기화 없음.** 사용자의 API 키, vault 내용, 프로젝트 파일은 운영자 서버로 전송되지 않습니다.
- **회원가입·계정 없음.** 핵심 키 관리 기능은 계정 없이 동작합니다. (※ 유료 라이선스 활성화 시 라이선스 키 검증은 Polar를 통해 처리됩니다 — [Privacy](./PRIVACY_ko.md) 참조.)
- **사용을 차단하지 않음.** KeyRadar는 게이트가 아니라 회전문입니다 — 누가 무엇을 했는지 *기록*하지만, 통제(누구한테 허용·차단)는 하지 않습니다. 그건 OS와 사용자의 몫입니다.
- **결제·청구를 막을 수 없음.** Provider의 결제 시스템 안에서 일어나는 일은 API 밖이라 직접 막지 못합니다. 사용량 급증을 빨리 알려 사용자가 콘솔에서 끊을 시간을 벌어주는 것까지가 한계입니다.
- **백도어식 복구 없음.** 마스터 패스워드를 잊으면 vault를 열 수 없습니다. 이 점은 "잠금이 진짜로 잠겨 있다"의 증거이지만, 사용자에게 위험이기도 합니다. 따라서 패스워드 설정 시 익스포트 백업을 강력히 권장합니다.

## 무료 체험 및 구매
- **무료 체험 3일** — 설치 후 3일간 모든 기능 사용 가능. 체험 기록은 사용자 기기에만 저장됩니다(완전 로컬). 동일 기기 재설치 흔적이 감지되면 체험은 1일로 단축됩니다(남용 방지). 이 제한은 OS 재설치·기기 변경 등으로 완벽하지 않을 수 있음을 명시합니다.
- **1회성 구매** — 구독이 아닙니다. 한 번 구매하면 라이선스 키가 발급되고, 일정 기간 업데이트를 받을 수 있습니다(자세한 기간은 [Terms](./TERMS_ko.md) 참조).
- **2대 기기까지 활성화** — 라이선스 키로 최대 2대 기기에 활성화 가능. 기기 변경 시 Polar Customer Portal에서 직접 활성화 인스턴스를 해제·재등록할 수 있습니다.
- **결제 처리** — Polar(merchant of record)가 결제와 라이선스 발급을 처리합니다.

## 보안 모델 요약
- **저장 시:** 항상 암호화. 평문 키가 디스크에 누워 있는 순간이 없습니다.
- **사용 시:** 키가 필요한 그 순간 메모리에서 잠깐 복호화됩니다.
- **공급 시(가장 약한 고리):** 평문이 짧게 존재합니다. 안전장치(자동 `.gitignore`, 타임아웃 삭제, 크래시 청소, 전수 로그)로 노출 창을 최소화합니다.
- **이동 시(다른 PC로):** 암호화된 vault를 통째로 복제합니다. 평문 USB 익스포트는 만들지 않습니다.

## 무인 자동화의 한계 (중요)
요즘 AI 에이전트는 사용자가 자리에 없을 때도 작동합니다. KeyRadar가 그 자동화의 길목을 막지 않으려면, 잠금이 자동으로 풀리는 상태(Always open 또는 세션 잠금 동안)가 필요합니다. **이때 KeyRadar의 암호화는 "기기 밖 유출"을 막지, "기기에 접근 가능한 무언가"는 막지 못합니다.** 이 한계를 숨기지 않고 사용자에게 명시합니다. 모르면 위험하기 때문입니다.

## 라이선스 및 책임 범위
KeyRadar는 사용자가 자신의 책임 하에 사용하는 도구입니다. KeyRadar는 비용 차단, 결제 방지, 키 유출 100% 방지를 약속하지 않습니다. 자세한 책임 한계는 [이용약관](./TERMS_ko.md)을 참고하세요.
