Legal, privacy, and license documents

KeyRadar is an AI provider API operations dashboard. It helps you discover, manage, verify, and monitor API keys locally. It is not a password manager or vault replacement.

Local first API keys, vault contents, and project files do not go to HAIB servers.
App-running scope Automatic verification and monitoring run only while KeyRadar is open.
Mac App Store Purchases are handled through Apple In-App Purchase.
