# Ripple Capture 개인정보 처리방침

**시행일:** 2026-07-19 (초안)
**버전:** 0.1 (Mac App Store 결제, 로컬 전용 캡처, 운영자 서버 없음, 변호사 검토 전)

## 한 줄 요약
Ripple Capture는 사용자의 **녹화 영상·스크린샷·GIF 및 화면 내용**을 운영자(HAIB) 또는 결제 인프라로 전송·저장하지 않습니다. 모든 캡처는 사용자 기기에서 로컬로만 처리됩니다. Mac App Store 결제는 Apple이 처리하며, Ripple Capture는 앱 잠금 해제에 필요한 로컬 라이선스 메타데이터만 저장합니다.

## 1. 본 방침의 범위
본 방침은 Ripple Capture 데스크톱 앱이 사용자의 데이터를 어떻게 다루는지 설명합니다.

## 2. 운영자가 직접 수집하는 정보
**없습니다.** HAIB는 자체 서버를 운영해 다음을 수집·전송·저장하지 않습니다.
- 사용자 식별 정보(이름, 이메일, IP 주소 등)
- 녹화 영상·스크린샷·GIF 등 캡처 결과물
- 화면 내용, 캡처 영역
- 사용량 통계
- 텔레메트리·분석 데이터
- 크래시 리포트(자동 전송 없음)
- 사용 패턴

## 3. 사용자 기기에 저장되는 정보
다음은 사용자의 로컬 기기에만 존재하며 외부로 전송되지 않습니다.

3.1 **캡처 결과물(영상·이미지·GIF)** — 사용자가 지정한 폴더(기본 `~/Movies/Ripple`)에 로컬 파일로만 저장됩니다. 운영자나 제3자에게 전송·업로드되지 않으며, 운영자는 접근할 수 없습니다.

3.2 **앱 환경설정** — 강조색, 언어, 커서 효과 세기, 저장 폴더 선택 등. macOS 표준 저장소(UserDefaults) 및 보안 스코프 북마크에 로컬 저장됩니다.

3.3 **최근 항목(히스토리)** — 최근 캡처 파일의 이름·경로 참조. 로컬에만 보관되며 파일 원본을 별도로 복제하지 않습니다.

3.4 **트라이얼 및 라이선스 메타데이터** — 라이선스 상태(`trial` / `expired` / `pro` 등), 트라이얼 시작 시각, 구매 정보. 로컬에 저장되며 사용자 기기 외부로 전송되지 않습니다.

## 4. 결제·라이선스·트라이얼에 관한 데이터
Ripple Capture는 Mac App Store 구매를 Apple을 통해 처리합니다.

4.1 **결제 시** — Apple App Store 결제 인프라가 결제 정보, 영수증, 거래 정보를 Apple 정책에 따라 처리합니다. HAIB는 카드 번호 등 결제 수단 정보를 받지 않습니다.

4.2 **라이선스 검증** — Ripple Capture는 Apple StoreKit 검증 결과를 수신하고, 그 결과로 얻은 로컬 entitlement 상태만 저장합니다.

4.3 **무료 체험 데이터** — 무료 체험 기록은 §3.4대로 로컬에만 저장됩니다. Ripple Capture는 트라이얼 강제를 위해 기기나 계정을 서버에서 추적하지 않습니다.

## 5. macOS 권한
Ripple Capture는 기능 수행을 위해 다음 권한을 요청할 수 있으며, 이는 실시간 로컬 처리에만 사용됩니다.

5.1 **화면 기록(Screen Recording)** — 화면을 녹화·캡처하기 위해 사용합니다. 캡처된 화면은 §3.1의 로컬 파일로만 저장되며 외부로 전송되지 않습니다.

5.2 **손쉽게 사용(Accessibility) — 선택** — 마우스 클릭 위치에 리플(물결) 효과를 그리기 위해 클릭 이벤트 발생을 감지합니다. **키 입력, 입력 내용, 클릭한 대상의 내용은 수집하지 않으며**, 효과 렌더링을 위한 좌표로만 실시간 사용됩니다. 이 권한 없이도 녹화는 동작하며, 클릭 리플 효과만 비활성화됩니다.

## 6. 외부로 나가는 통신
Ripple Capture가 외부와 통신하는 경우는 **Apple App Store 인프라(구매·복원·StoreKit 검증)** 로 한정됩니다. 이 외에는 외부 통신이 없습니다 — 사용 통계 전송, 원격 로그, 자체 서버 통신을 수행하지 않습니다.

## 7. 데이터 삭제·이전
7.1 **삭제:** 캡처 결과물은 사용자가 Finder에서 직접 관리·삭제합니다. 앱 환경설정은 앱 제거 또는 macOS 표준 방식으로 삭제됩니다. Apple 측 구매 기록은 Apple을 통해 관리됩니다.

7.2 **앱 제거:** 앱을 제거해도 사용자가 저장한 캡처 파일은 지정 폴더에 그대로 남습니다(사용자 자료 보호). 완전 삭제는 해당 파일을 직접 삭제하십시오.

## 8. 어린이 개인정보
Ripple Capture는 일반 사용자를 대상으로 하며, 13세 미만(또는 관할 법령상 아동)의 개인정보를 의도적으로 수집하지 않습니다.

## 9. 사용자의 권리 (GDPR/한국 개인정보보호법 등)
- **HAIB 보유 정보:** HAIB는 사용자 개인정보를 자체 서버에 보유하지 않습니다.
- **Apple 보유 정보:** 구매 및 거래 정보는 Apple을 통해 관리됩니다.
- **로컬 데이터:** 사용자가 직접 관리합니다.

## 10. 보안 사고 통지
